# Datenschutzerklärung

AvoCook sammelt keine Daten für mich und enthält weder Werbung noch Analyse-Tools.

Im lokalen Modus bleiben Rezepte, Bilder und Einstellungen auf dem Gerät.

Wenn der Benutzer einen Nextcloud-Server verbindet, werden die Rezepte direkt zwischen der App und diesem Server synchronisiert. Die Anmeldedaten werden über den Expo SecureStore im sicheren Speicher des Systems abgelegt.

Beim Importieren von einer URL wird die angeforderte Seite heruntergeladen, um öffentlich zugängliche Rezeptdaten zu lesen.
