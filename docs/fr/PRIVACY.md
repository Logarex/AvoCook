# Politique de confidentialité

AvoCook ne collecte aucune donnée à mon intention et n'inclut ni publicité ni outils d'analyse.

En mode local, les recettes, les images et les paramètres restent sur l'appareil.

Si l'utilisateur connecte un serveur Nextcloud, les recettes sont synchronisées directement entre l'application et ce serveur. Les identifiants sont stockés dans le stockage sécurisé du système via Expo SecureStore.

L'importation depuis une URL télécharge la page demandée afin de lire les données de recettes disponibles publiquement.
