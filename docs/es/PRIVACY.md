# Política de privacidad

AvoCook no recopila datos para mí y no incluye anuncios ni análisis.

En el modo local, las recetas, las imágenes y la configuración permanecen en el dispositivo.

Si el usuario conecta un servidor Nextcloud, las recetas se sincronizan directamente entre la aplicación y ese servidor. Las credenciales se almacenan en el almacenamiento seguro del sistema a través de Expo SecureStore.

La importación desde una URL descarga la página solicitada para leer datos de recetas disponibles públicamente.
