# Informativa sulla privacy

AvoCook non raccoglie dati per me e non include annunci pubblicitari o strumenti di analisi.

In modalità locale, ricette, immagini e impostazioni rimangono sul dispositivo.

Se l'utente collega un server Nextcloud, le ricette vengono sincronizzate direttamente tra l'app e quel server. Le credenziali vengono memorizzate nella memoria sicura del sistema tramite Expo SecureStore.

L'importazione da un URL scarica la pagina richiesta per leggere i dati delle ricette disponibili al pubblico.
