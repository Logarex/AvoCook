## AvoCook v2.0 - Shopping List, Animations and New Languages
English Version

This major update brings highly anticipated features, a refined interface, and streamlined distribution to make installing the app easier than ever!

### Download & Installation

- **iOS (iPhone / iPad)**: Download on the App Store
- **Android (APK)**: Download the avocook.apk file from the assets section below and open it to install the update. Your local recipes will remain completely safe thanks to our unique EAS signing key.
- **F-Droid**: Coming soon via the IzzyOnDroid repository.

### What's New in v2.0.0

- **Integrated Shopping List**: A brand-new interactive tab with local persistent storage to manage your ingredients directly within the app.
- **Auto-Update & SQLite Safety (Android)**: A lightweight system now checks at startup if a new APK is available on GitHub and displays a banner. If you are using local mode, the app will safely advise you to create a backup before updating.
- **Transition Animations**: Added smooth directional slide animations when switching between tabs.
- **New Languages**: Full support for Spanish (ES) and Italian (IT), joining French, English, and German.
- **Localized Import Placeholders**: URL import suggestions now automatically adapt to your device's language (e.g., Marmiton for French, GialloZafferano for Italian, etc.).
- **Duplicate Prevention**: Recipe duplication is now a thing of the past. The app detects and prevents duplicates.

### Improvements & Fixes

- **Nutritional Data**: Full support and numerical value normalization for improved nutrient management (Closes #39).
- **Edit Mode**: The top back button is now hidden while editing a recipe to free up visual space.
- **Cleanup & Security**: Removed untracked sensitive configuration files and improved .gitignore rules.
